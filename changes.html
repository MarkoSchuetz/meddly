New in this branch

Simple interface

  • Added const to array of terminal values, in parameter to createEdgeForVar methods. This should have minimal, if any, impact on applications.
  • Removed const from arrays in parameters to createEdge methods, as the implementation will now reorder these arrays in place as needed for speed. If you expect these arrays to remain untouched, then you will need to copy them before calling the method.
  • Added forest constants DONT_CARE and DONT_CHANGE, which should be used instead of raw values, for calls to createEdge() methods.
